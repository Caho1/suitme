# Suitme - 数字模特图像生成服务

Suitme 是一个 AI 驱动的数字模特图像生成后端服务，为电商穿搭展示提供支持。

## 核心功能

1. **默认模特生成 (F1)** - 根据用户照片和身体参数生成标准数字模特形象
2. **模特编辑 (F2)** - 在已有模特基础上进行局部调整（发型、身材等）
3. **穿搭生成 (F3)** - 将服装单品"穿"到数字模特上，支持 front/side/back 视角

## 服务定位

- 内部服务，仅面向 Java 业务后端，不直接对外暴露
- 异步任务模式：创建任务 → 轮询状态 → 回调结果
- 使用 Apimart Images API 进行实际的 AI 图像生成
